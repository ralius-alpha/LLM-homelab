## シーズン1: 基礎構築編

### 第1部: 基礎構築（1〜3話）

#### 【第1話】序章：なぜ今、自宅にAIを置くのか

- 連載全体の見取り図
- ローカルLLMとクラウドAIの違い
- 「次世代エージェント」とは何か
- 必要なPCスペック目安（クラスA/B/C分類）
- **今月のゴール**: 自分のPC環境を把握する
- **次号予告**: いよいよAIを召喚

#### 【第2話】最初のローカルAIを召喚する

- Ollama / LM Studio インストール
- 最初のモデルダウンロード〜対話
- GPU/CPU動作確認方法
- **今月のゴール**: ターミナルで日本語対話成功
- **次号予告**: ブラウザから使えるようにする

#### 【第3話】ブラウザUIで快適に使う

- Open WebUI 導入
- チャット履歴・モデル切替・システムプロンプト
- 複数モデル使い分け（軽量/高性能/Vision）
- **今月のゴール**: ChatGPT風UIで自分のAIと対話
- **次号予告**: 外出先から使えるようにする

---

### 第2部: 外部アクセス・API（4〜6話）

#### 【第4話】どこからでもアクセス：Cloudflare Tunnel

- なぜポート開放しないのか（セキュリティ観点）
- Cloudflare Tunnel セットアップ
- 認証レイヤーの追加（Cloudflare Access）
- **今月のゴール**: スマホから自宅AIにアクセス
- **次号予告**: APIとして叩く

#### 【第5話】APIで叩く・自作スクリプトから使う

- Ollama の OpenAI 互換API
- Python から呼ぶサンプル
- 簡単な自動化スクリプト例
- **今月のゴール**: 自作コードからAIを呼べる
- **次号予告**: AIに「見る目」を与える

#### 【第6話】マルチモーダル：画像を理解させる

- Vision対応モデル導入（Qwen2.5-VL 等）
- 写真の説明・OCR・図表読み取り
- プライベート活用アイデア
- **今月のゴール**: 撮った写真をAIに解説させる
- **次号予告**: AIに「耳」を与える

---

### 第3部: エージェント化（7〜9話）

#### 【第7話】音声入力：Whisperで文字起こし

- Faster-Whisper 導入
- 音声メモ・動画の文字起こし
- LLMと組み合わせて要約自動化
- **今月のゴール**: 音声メモが自動でテキスト化＆要約
- **次号予告**: AIに「記憶」を与える

#### 【第8話】RAG入門：個人ナレッジベース構築

- ベクトルDB（Chroma 等）の概念
- 自分のメモ・読書記録を食わせる
- 「自分専用に育ったAI」の第一歩
- **今月のゴール**: 自分のデータに基づいて答えるAI
- **次号予告**: AIに「手足」を与える

#### 【第9話】Function Calling：AIに道具を持たせる

- Function Calling の仕組み
- 簡単なツール実装
- 「エージェント」の本質的な体験
- **今月のゴール**: AIが自分で道具を選んで使う体験
- **次号予告**: ノーコードで組む

---

### 第4部: 応用・カスタマイズ（10〜12話）

#### 【第10話】Dify / n8n でノーコード構築

- ワークフロー型AIアプリの設計
- 複数AIを連携させる
- プライベート用途のテンプレ集
- **今月のゴール**: 自分専用ワークフロー1個完成
- **次号予告**: モデルを育てる

#### 【第11話】モデルカスタマイズ入門

- システムプロンプト設計
- Modelfile（Ollama）でキャラ作り
- 軽量ファインチューニングの概念
- **今月のゴール**: 自分好みのAIキャラを作る
- **次号予告**: 集大成

#### 【第12話】総集編：1年間で得たもの・次のステップ

- 連載全体の振り返り
- 読者構成事例紹介（コメント募集）
- 最新ローカルAI動向
- 次の学習リソース
- **今月のゴール**: 自分の理想構成を言語化
- **次号予告**: シーズン2始動
